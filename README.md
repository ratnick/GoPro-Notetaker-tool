# GoPro Notetaker tool

Control GoPro 3/4/5/6 camera from a PC/Mac and synchronize notes via timestamps in the video. For professional in-field use.

## Getting Started

See the file "ViNo Get Started DEVELOPMENT V4.0.pptx"

### Prerequisites

IDE used: Eclipse and SceneBuilder
Developed on PC Win7. Original development on Mac.
